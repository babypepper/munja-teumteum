# 문자틈틈

문자틈틈은 자주 보내야 하는 문자 메시지를 미리 준비해두고, 원하는 날짜와 시간에 예약 발송할 수 있는 Android 앱입니다.

반복적으로 안부 문자, 안내 문자, 확인 문자를 보내야 할 때 틈틈이 챙겨 보낼 수 있도록 만든 앱입니다.

## 다운로드

최신 APK는 아래 Releases 페이지에서 받을 수 있습니다.

https://github.com/babypepper/munja-teumteum/releases/latest

현재 정식 배포 버전:

- v1.1.2
- 파일명: munja-teumteum-v1.1.2-release.apk

## 주요 기능

- 문자 예약 발송
- 저장된 문구와 번호를 골라 빠른 예약 추가
- 한 통 예약
- 월간 랜덤 예약
- 09:30~20:30 사이 자동 분산 예약
- 전체 / 발송 / 예약 내역 확인
- 예약 수정 및 취소
- SMS 권한과 배터리 제한 상태 확인
- 문자틈틈 보관함 파일 사용
- WIFI를 켜두어도 녹색문자로 발송

## 설치 방법

1. Releases 페이지에서 최신 APK 파일을 다운로드합니다.
2. Android 휴대폰에서 APK 파일을 엽니다.
3. 설치 중 “알 수 없는 앱 설치” 안내가 나오면 허용합니다.
4. 설치 후 문자 발송 권한을 허용합니다.
5. 안정적인 예약 발송을 위해 배터리 제한 해제를 권장합니다.

## 사용 방법

1. 앱을 실행합니다.
2. 문자 발송 권한을 허용합니다.
3. 설정에서 배터리를 제한없음으로 변경합니다.
4. 문구보관함에서 샘플 200개 추가를 선택합니다.
5. 번호보관함에서 번호추가를 눌러서 문자를 보낼 번호를 여러개 적습니다.
6. `+ 새 예약 만들기`를 눌러 예약을 추가합니다.
7. 한 통 예약 또는 월간 랜덤 예약을 선택합니다.
8. 예약 내역에서 발송 예정 문자를 확인합니다.

## 월간 랜덤 예약

월간 랜덤 예약은 선택한 기간 동안 하루 한 통씩 문자를 예약합니다.

예약 시간은 09:30~20:30 사이에서 자동으로 분산됩니다.  
이미 예약이 있거나 지난 날짜는 건너뜁니다.

## 주의사항

- 휴대폰 전원이 꺼져 있거나 통신 상태가 불안정하면 예약 문자가 발송되지 않을 수 있습니다.
- 일부 기기에서는 배터리 최적화 때문에 예약 발송이 지연될 수 있습니다.
- 안정적인 사용을 위해 문자틈틈 앱의 배터리 제한을 해제하는 것을 권장합니다.
- 문자 발송 요금은 사용 중인 통신사 요금제에 따라 부과될 수 있습니다.
